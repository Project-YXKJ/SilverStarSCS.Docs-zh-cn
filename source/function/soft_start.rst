软启动
======

**功能**

- 在一段新线迹开始时
- 速度被限制在 :option:`S08` 之内
- 和前加固并行发生时，一般速度要比加固速度更低
- 调速器松开时可以中断，即可以停车中断
- 所需针数可以调过 :option:`O01` 来设置
- 全后踩即被终止，即剪线后重置

快速参考
--------

下表总结了软启动功能所使用到的参数：

========== ====== =============
参数       权限   参见
========== ====== =============
软启动     操作员 :option:`A21`
软启动速度 技术员 :option:`S08`
软启动针数 技术员 :option:`O01`
========== ====== =============

参数列表
--------

.. option:: A21

    -Max  1
    -Min  0
    -Unit  --
    -Description
      | 软起动：
      | 0 = 关闭；
      | 1 = 打开

.. option:: S08

    -Max  1000
    -Min  50
    -Unit  spm
    -Description  软起动速度

.. option:: O01

    -Max  10
    -Min  1
    -Unit  针
    -Description  软起动生效的针数
