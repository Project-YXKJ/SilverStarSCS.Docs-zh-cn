缝制功能
========

**缝纫功能** 专注于提供有关特定主题的全面信息。在这里您可以学习如何设置参数来实现您想要的功能。

什么是自动化规则？
==================

规则允许您根据您设置的标准在系统内自动执行操作，自动化规则由三部分组成：

* **触发器**
* **条件**
* **动作**
