# YSC-90A0

## 规格概要

- 下挂独立控制箱；
- 交流 220v±10%；
- 10 路电磁铁（阀）输出；

## 背板接口

```{image} ../_static/90a0/backcover.excalidraw.svg
:scale: 100 %
:align: center
```

| 标签  | 含义             |
| ----- | ---------------- |
| M1    | 主轴电机电源     |
| M1.1  | 主轴电机码盘     |
| PANEL | 触摸屏           |
| PEDAL | 调速器           |
| X100  | 扩展信号端口     |
| X101  | 信号端口         |
| X200  | 电磁铁（阀）输出 |
